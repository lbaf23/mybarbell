# mybarbell

- flutter
- material design
- cross-platform


## Getting Started

- flutter

## Development

- generate l10n

```bash
flutter gen-l10n
```

## Build

- build for android

```bash
flutter build apk --release
```

- build for windows

```bash
flutter build windows --release
```
